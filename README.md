## Web проект ReleaseBox (RB) - 

### Запуск проекта: 
backend: 
- /backend  

- /python -m venv venv 

- /venv/Scripts/activate 

- pip install -r requirements.txt 

- cd /releasebox 

- python manage.py runserver 

frontend: 

- /frontend/rb-project 

- yarn 

- yarn dev
